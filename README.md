# 📘 Mentor IA API Documentation

Welcome to the official documentation for the **Mentor IA API**.  
This guide is built with [Mintlify](https://mintlify.com/) and provides developers with everything needed to understand and integrate with the API.

It covers core modules including:

- ✅ My Assistant  
- 👤 Accounts  
- 🧠 Knowledge Base  
- 📄 Content


## 🚀 Getting Started

To run the documentation locally, make sure you have the following installed:

- [Node.js](https://nodejs.org/)
- [Mintlify CLI](https://mintlify.com/docs/cli)



## 🛠️ Local Development

Follow these steps to run the documentation on your machine:

1. **Install the Mintlify CLI** (if you haven't already):
   ```bash
   npm install -g mintlify
   ```

2. **Start the Mintlify dev server**:

   ```bash
   mint dev
   ```

3. **View the documentation** in your browser at:

   ```
   http://localhost:3000
   ```
